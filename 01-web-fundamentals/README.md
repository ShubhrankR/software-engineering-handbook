# Web Fundamentals

This module explains what happens from the moment a user enters a URL in the browser until the webpage is rendered on the screen.

## Chapters

- ✅ DNS Resolution
- ✅ TCP & IP
- ⬜ TLS & HTTPS
- ⬜ HTTPD
- ⬜ Browser Request Lifecycle
- ⬜ HTML Parser & DOM
- ⬜ CSSOM
- ⬜ Render Tree
- ⬜ Layout
- ⬜ Paint
- ⬜ Composite

## Learning Outcome

After completing this module, I should be able to explain the complete lifecycle of a web request—from typing a URL in the browser to rendering pixels on the screen.
